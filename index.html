Built for speed

AngularFire is built to make sure you can prototype your app as quickly as possible without having to set up servers or write any backend code.

3-way data binding

Saving user or app data is a breeze with AngularFire's 3-way data binding. It's never been easier to extend Angular's quintessential 2-way data binding to the server.

Flexible API

Choose between an easy to use implicit sync API that will automatically keep all your Angular models persisted and updated, or a more powerful explicit sync API that gives you control over when data gets synchronized.
